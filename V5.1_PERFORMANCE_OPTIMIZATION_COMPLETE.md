## 核心技术实现

### 1. 数据层架构

```
用户请求
    ↓
IndexedDB缓存检查
    ↓
缓存命中? → 是 → 立即返回（<50ms）
    ↓ 否
ChunkLoader分块加载
    ↓
保存到IndexedDB
    ↓
返回数据
    ↓
后台检查更新
```

### 2. 搜索架构

```
用户输入
    ↓
搜索缓存检查
    ↓
缓存命中? → 是 → 立即返回
    ↓ 否
倒排索引搜索
    ↓
TF-IDF排序
    ↓
保存到缓存
    ↓
返回结果（<100ms）
```

### 3. 渲染架构

```
文献数据
    ↓
虚拟滚动计算可见范围
    ↓
从对象池获取DOM节点
    ↓
批量更新DOM
    ↓
requestAnimationFrame渲染
    ↓
60fps流畅显示
```

### 4. 离线架构

```
用户访问
    ↓
Service Worker拦截
    ↓
静态资源? → 是 → Cache First
    ↓ 否
数据文件? → 是 → Network First
    ↓
返回响应
    ↓
更新缓存
```

## 已知限制

1. **IndexedDB限制**
   - 浏览器配额限制（通常>50MB）
   - 需要用户授权（首次使用）
   - 不支持IE浏览器

2. **Service Worker限制**
   - 仅支持HTTPS（localhost除外）
   - 首次访问无法离线
   - 更新需要刷新页面

3. **虚拟滚动限制**
   - 需要固定或可预测的项高度
   - 动态高度会影响性能
   - 不适合复杂嵌套结构

4. **搜索限制**
   - 中文分词较简单
   - 不支持拼音搜索
   - 索引占用内存
