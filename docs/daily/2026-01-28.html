01

今日摘要

总览：今日文献中，交替磁性（altermagnetism）与非常规超导的交叉占据显著位置，涵盖CrSb中自旋分裂费米面的量子振荡、MnF₂的多极序弹性热探测，以及Eu掺杂无限层镍酸盐中顺磁驱动的超导重入；机器学习势与神经网络方法持续渗透材料发现与光谱解析，包括通过机器学习加速发现巨极化超四方钙钛矿、MAD-SURF势描述贵金属表面分子吸附，以及融合多模态光谱揭示局域结构。

热点：1. 交替磁性 → 非常规自旋-轨道耦合与输运现象 → CrSb量子振荡证实自旋分裂费米面、MnF₂应变下弹性热效应探测多极序、二维Janus铁电交替磁体实现全电控自旋纹理切换。2. 无限层镍酸盐超导 → 电子关联、顺磁环境与绝缘-超导转变的调控 → Eu掺杂SmNiO₂中顺磁驱动超导重入、无限层钐镍酸盐中超导-绝缘转变的玻色相。3. 机器学习嵌入实验光谱与原子模拟 → 势函数构建、结构反演与动力学加速 → 超四方钙钛矿巨极化的机器学习加速发现、STM设定点依赖谱的分子哈密顿学习、GaN辐照损伤的机器学习分子动力学。
