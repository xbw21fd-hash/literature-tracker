今日摘要

总览：今日文献中，量子几何与平带物理形成显著聚类：量子几何偶极子驱动铁磁序[2]、量子几何标识平带不稳定性[34]、量子度规修正刘维尔定理与手征动理学[45]及量子度规定域化[24]共同指向量子几何张量在关联电子现象中的核心角色。超导领域涵盖莫尔WSe₂双层中的 gossamer 超导[21]、非中心对称超导体CeRh₂As₂的涡旋态[46]、受挫自旋纹理诱导的三重态约瑟夫森二极管效应[50]及PtBi₂的点接触增强超导[42]，显示非常规配对机制与自旋-轨道耦合的多元探索。机器学习方法持续渗透：图神经网络用于格点规范论的威尔逊圈表示[3]、分子表面自组装分类[12]，以及主动学习加速过渡路径采样[10]和MLIP训练[7]。

热点：1. 量子几何作为平带关联效应的统一语言 → 将量子度规与贝里曲率从不锈钢色散关系中的几何量提升为决定磁性、超导不稳定性及输运性质的物理驱动力 → 典型工作包括量子几何偶极子诱导的味铁磁[2]、平带系统中用量子几何识别不稳定性[34]及量子度规对动理学方程的修正[45]。2. 主动学习与神经势能面（MLIP）的深度耦合 → 通过不确定性量化或预训练表示指导原子构型采样，降低数据成本并扩展模拟时长 → 典型工作包括基于过渡路径采样的主动学习发现反应机理[10]、利用预训练模型表示作为主动学习采集信号[7]。3. 受挫磁性/自旋纹理与非常规超导的交叉 → 非共线自旋序通过自旋-轨道耦合诱导三重态配对、约瑟夫森二极管效应及空间非均匀超导态 → 典型工作包括受挫自旋纹理导致的三重态约瑟夫森耦合[29]与二极管效应[50]、Majorana平带诱导的非均匀超导[23]。
