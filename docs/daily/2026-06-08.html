热点：机器学习原子尺度建模 → 从等变原子间势、多目标优化、量子杂化到长程核函数压缩和晶体图网络输运预测 → 典型条目包括等变深度学习原子间势、长椭球波函数长程势、晶体结构到热电输运多尺度图神经网络。可解释物理机器学习 → 把外复标度、残差校准不确定度、物理约束神经网络、傅里叶神经算子与成像数据耦合，用于量子散射、洪水反演、仿星器湍流和四维变分同化 → 典型条目包括外复标度量子散射网络、无害物理约束逆学习、温德尔施泰因七型电子尺度湍流建模。低维自旋与铁电界面工程 → 通过詹纳斯单层、稀土替位、氧化物界面原子级调控、衬底极化面和外延取向改变各向异性、交换作用、畴结构与自旋霍尔磁阻符号 → 典型条目包括铬硫卤单层铁磁、钕替位钆钒锡笼目金属、氧化物异质结构自旋霍尔磁阻变号、氮化铝衬底上少层碘化铬异常磁序。强关联与拓扑量子临界 → 围绕笼目超导、多能带莫尔体系、非厄米自旋链、退禁闭量子临界和量子自旋冰热探针，关注量子度量非线性霍尔响应、表面平带、质量增强与反常边界条件 → 典型条目包括铼锆多能带超导、石墨烯莫尔超晶格金属绝缘体转变、三维超导交错磁体交叉表面平带、张量网络自旋声子链。
