今日摘要

总览：今日文献中，量子材料与自旋电子学占据核心位置：自主自旋波光谱的混合主动学习、射频远程编程自旋神经网络权重、以及费米面机器学习分析共同指向数据驱动的磁学表征新范式。同时，张量网络方法（广义置信传播收缩、自适应采样量子最优控制）与神经网络量子态（双曲循环神经网络构建非欧几里得态）持续扩展量子多体可计算边界，而U(1)量子自旋液体的动力学制备与受挫量子材料中的涌现自旋超固体则代表了模拟量子计算与强关联物理的前沿实验方向。

热点：1. 量子材料表征的自动化与智能化 → 将主动学习、神经网络与实验闭环结合以减少测量成本 → 典型工作[1]混合主动学习用于自主自旋波光谱、[3]机器学习加速费米面分析以筛选自旋电子材料；2. 物理启发的神经网络架构向量子与经典多体系统深度渗透 → 图神经网络用于晶体结构预测与晶格哈密顿量推断、神经算子学习参数依赖谱、以及物理信息神经网络校正湍流模型 → 典型工作[8]哈密顿图推断网络联合发现结构与预测动力学、[11]物理信息算子学习参数依赖谱、[16]将科学知识嵌入神经网络密度泛函；3. 量子自旋液体与拓扑关联态的制备、表征与新奇激发 → 从模拟器中的U(1)量子自旋液体动力学制备到受挫磁体中的自旋超固体，再到BCS波函数的量子多体疤痕性质 → 典型工作[6]模拟量子模拟器动力学制备U(1)量子自旋液体、[31]特定BCS波函数具有量子多体疤痕特征、[46]受挫量子材料中涌现自旋超固体。
