热点：热点一：机器学习势与神经算子加速多尺度模拟 → 用可迁移深度势处理熔融盐、机器学习模型模拟非紧致棱柱位错环动力学、Fourier 神经算子做海啸传播与 PDE 代理 → 代表工作为 LiF-BeF2 深度势、位错环 ML 模拟、jNO 库与离岸海啸 FNO 反演。热点二：强关联体系的张量网络与量子蒙特卡洛新方法 → iPEPS 高精度刻画 Shastry-Sutherland 自旋液体、零模规范固定截断有限温 Z2 格点规范、张量网络求解非周期超 moiré 动量分辨谱、后测量 QMC → 体现于条目 [43][35][39][50]。热点三：量子化学中的神经网络波函数与资源削减 → 神经网络量子态的确定性训练框架、面向选择组态相互作用的 learning-to-rank、相关均场降维以减少量子中心超算资源、块结构矩阵乘法加速局域积分 → 见条目 [5][14][52][38]。此外，自旋电子学方向出现自旋弹性新范式、手性诱导磁非互易及 STT-MRAM 磁化对齐等实验-理论结合工作。
