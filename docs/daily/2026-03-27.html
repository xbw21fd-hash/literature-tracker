01

今日摘要

总览：今日文献呈现量子多体计算与机器学习深度融合的密集态势：张量回流、神经网络量子态、深度量子蒙特卡洛等方法被用于费米-哈伯德模型与强关联体系的基态与动力学模拟；同时，神经算子量子态、机器学习增强无轨道密度泛函理论、以及镁-铝-硅-氧和铬-钴-镍合金的通用机器学习势，正推动第一性原理精度向大尺度材料模拟延伸。弛豫铁电体本征结构、莫尔二硒化钨中的压力增强关联磁性及陈绝缘态、以及氢穿透石墨炔的量子隧穿效应等具体材料问题亦有新进展。

热点：1. 量子嵌入与变分方法的神经网络替代 → 以神经网络替代杂质求解器或构建低成本的量子态代理模型，降低多体计算成本 → 典型工作包括神经网络作为量子嵌入杂质求解器的替代体（[2]）、神经算子量子态作为量子动力学基础模型（[6]）、以及利用拉格朗日技术实现从头算变分蒙特卡洛中原子力的快速无偏评估（[13]）。 2. 机器学习势与第一性原理方法的规模化拓展 → 通过经典-量子迁移学习、可迁移深度量子蒙特卡洛或通用机器学习势，实现高温高压地球材料、高熵合金及强关联体系的大尺度模拟 → 典型工作涵盖镁-铝-硅-氧体系的通用机器学习势（[22]）、铬-钴-镍合金的通用机器学习势（[27]）、以及可迁移深度量子蒙特卡洛对强关联体系几何优化的启用（[29]）。 3. 量子-经典机器学习在物理问题中的方法学交叉 → 量子核方法、强化学习处理含记忆量子过程、以及谱方法在量子计算与机器学习间的桥梁作用 → 典型工作包括资源高效量子核（[10]）、量子-经典支持向量机的核级比较（[15]）、以及针对含记忆量子过程的强化学习（[3]）。
