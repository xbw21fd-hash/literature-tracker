## 🚀 技术架构

### 分层缓存
```
用户请求
    ↓
内存缓存 (L1) - 最快
    ↓
IndexedDB (L2) - 持久化
    ↓
Service Worker (L3) - 离线
    ↓
网络请求 (L4) - 在线
```

### 搜索流程
```
用户输入
    ↓
倒排索引查询
    ↓
TF-IDF排序
    ↓
结果过滤
    ↓
虚拟滚动渲染
```

### 性能监控
```
Performance API
    ↓
PerformanceMonitor
    ↓
实时面板 + 报告
    ↓
优化建议
```
