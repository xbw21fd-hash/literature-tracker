## 🔧 集成方式

### 1. 自动初始化

系统在页面加载时自动初始化所有性能优化功能:

```javascript
document.addEventListener('DOMContentLoaded', async () => {
    if (typeof initPerformanceOptimization === 'function') {
        await initPerformanceOptimization();
    }
});
```

### 2. IndexedDB缓存

首次加载时从网络获取数据并缓存到IndexedDB,后续访问优先从缓存加载:

```javascript
// 尝试从缓存加载
const cachedArticles = await indexedDBManager.getAllArticles();
if (cachedArticles && cachedArticles.length > 0) {
    articles = cachedArticles;
    fromCache = true;
}

// 后台检查更新
if (fromCache) {
    checkForUpdates();
}
```

### 3. 倒排索引搜索

搜索时优先使用倒排索引引擎:

```javascript
if (searchTerm && invertedIndexSearchEngine) {
    filteredArticles = invertedIndexSearchEngine.search(searchTerm);
} else {
    // 回退到原有搜索逻辑
    filteredArticles = advancedSearch(filteredArticles, searchTerm);
}
```

### 4. 性能监控

点击右下角⚡按钮打开性能监控面板:

```javascript
performanceMonitor.showPanel();
```
