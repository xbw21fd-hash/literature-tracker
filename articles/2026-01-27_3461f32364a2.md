## Abstract
arXiv:2601.17282v1 Announce Type: new Abstract: Inference is a versatile tool that underlies scientific discovery, machine learning, and everyday decision-making: it describes how an agent updates a probability distribution as partial information is acquired from multiple measurements, reducing ignorance about a system's latent state. We define an inferential efficiency as the ratio of information gain to cumulative memory erasure cost, with inefficiency arising from unexploited correlations between the measured system and memories, and/or between memories and environment (noise). Using this efficiency, we benchmark two limiting measurement paradigms: sequential, in which the same memory is exploited iteratively, and parallel, in which many memories are exploited simultaneously. In both cases, the minimal erasure cost reflects correlations across memories: temporal in sequential, spatial in parallel. Remarkably, when all system-memory correlations are exploited for inference, both paradigms attain the same minimal erasure cost, even in the presence of noise. Conversely, the parallel paradigm performs better in the presence of unexploited correlations, stemming from hidden memories' degrees of freedom. This approach provides a quantitative, physically grounded criterion to compare inference strategies, determine their efficiency, and link target information gains to their minimal entropic cost.

## 摘要（中文）
arXiv:2601.17282v1 公告类型：新 摘要：推理是一种多功能工具，是科学发现、机器学习和日常决策的基础：它描述了代理如何在从多个测量中获取部分信息时更新概率分布，从而减少对系统潜在状态的无知。我们将推理效率定义为信息增益与累积内存擦除成本的比率，效率低下是由于测量系统与内存之间和/或内存与环境（噪声）之间未利用的相关性引起的。利用这种效率，我们对两种限制性测量范例进行了基准测试：顺序（迭代地利用相同的内存）和并行（同时利用许多内存）。在这两种情况下，最小擦除成本反映了记忆之间的相关性：时间顺序，空间并行。值得注意的是，当利用所有系统内存相关性进行推理时，即使存在噪声，两种范例也能获得相同的最小擦除成本。相反，由于隐藏记忆的自由度，并行范式在存在未利用的相关性时表现更好。这种方法提供了一个定量的、基于物理的标准来比较推理策略，确定其效率，并将目标信息增益与其最小熵成本联系起来。
