## Abstract
arXiv:2602.03546v1 Announce Type: cross Abstract: Machine learning is a powerful method of extracting meaning from data; unfortunately, current digital hardware is extremely energy-intensive. There is interest in an alternative analog computing implementation that could match the performance of traditional machine learning while being significantly more energy-efficient. However, it remains unclear how to train such analog computing systems while adhering to locality constraints imposed by the physical (as opposed to digital) nature of these systems. Local learning algorithms such as Equilibrium Propagation and Coupled Learning have been proposed to address this issue. In this paper, we develop an algorithm to exactly calculate gradients using a graph theoretic and analytical framework for Kirchhoff's laws. We also introduce Generalized Equilibrium Propagation, a framework encompassing a broad class of Hebbian learning algorithms, including Coupled Learning and Equilibrium Propagation, and show how our algorithm compares. We demonstrate our algorithm using numerical simulations and show that we can train resistor networks without the need for a replica or readout over all resistors, only at the output layer. We also show that under the analytical gradient approach, it is possible to update only a subset of the resistance values without a strong degradation in performance.

## 摘要（中文）
arXiv:2602.03546v1 公告类型：cross 摘要：机器学习是一种从数据中提取意义的强大方法；不幸的是，当前的数字硬件非常耗能。人们对替代模拟计算实现很感兴趣，它可以与传统机器学习的性能相匹配，同时显着提高能源效率。然而，目前尚不清楚如何训练此类模拟计算系统，同时遵守这些系统的物理（而不是数字）性质所施加的局部性限制。人们提出了平衡传播和耦合学习等局部学习算法来解决这个问题。在本文中，我们开发了一种算法，使用基尔霍夫定律的图论和分析框架来精确计算梯度。我们还介绍了广义均衡传播，这是一个包含广泛的赫布学习算法的框架，包括耦合学习和均衡传播，并展示了我们的算法如何进行比较。我们使用数值模拟演示了我们的算法，并表明我们可以训练电阻网络，而无需仅在输出层复制或读出所有电阻。我们还表明，在分析梯度方法下，可以仅更新电阻值的子集，而不会严重降低性能。
