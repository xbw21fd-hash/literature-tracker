## Abstract
arXiv:2601.17485v1 Announce Type: new Abstract: We formulate a sparse-to-dense reconstruction layer for fractured media in which sparse point measurements are mapped onto piecewise-planar fracture supports inferred from 3D trace polylines. Each plane is discretized in local coordinates and estimated via a convex objective that combines a grid SPDE/GMRF quadratic prior with an $\ell_1$ penalty on undecimated discrete curvelet coefficients, targeting anisotropic, fracture-aligned structure that is poorly represented by isotropic smoothness alone. We further define an along-fracture distance through trace-network geodesics and express connectivity-driven regularization as a quadratic form $z^\top P^\top L_G P z$, where $L_G$ is a graph Laplacian on the trace network and $P$ maps plane grids to graph nodes; plane intersections are handled by linear consistency constraints sampled along intersection lines. The resulting optimization admits efficient splitting: sparse linear solves for the quadratic block and coefficient-wise shrinkage for the curvelet block, with standard ADMM convergence under convexity. We specify reproducible synthetic benchmarks, baselines, ablations, and sensitivity studies that isolate directional sparsity and connectivity effects, and provide reference code to generate the figures and quantitative tables.

## 摘要（中文）
arXiv:2601.17485v1 公告类型：新 摘要：我们为裂缝介质制定了一个从稀疏到密集的重建层，其中稀疏点测量被映射到从 3D 迹线折线推断的分段平面裂缝支撑上。每个平面在局部坐标中离散化，并通过凸目标进行估计，该凸目标将网格 SPDE/GMRF 二次先验与对未抽取离散曲波系数的 $\ell_1$ 惩罚相结合，针对仅由各向同性平滑度很难表示的各向异性、断裂对齐结构。我们进一步通过追踪网络测地线定义沿断裂距离，并将连接驱动的正则化表达为二次形式 $z^\top P^\top L_G P z$，其中 $L_G$ 是追踪网络上的图拉普拉斯算子，$P$ 将平面网格映射到图节点；平面相交由沿相交线采样的线性一致性约束处理。由此产生的优化允许有效的分割：二次块的稀疏线性求解和曲线块的系数收缩，以及凸性下的标准 ADMM 收敛。我们指定可重复的综合基准、基线、消融和敏感性研究，以隔离方向稀疏性和连通性影响，并提供参考代码来生成图形和定量表。
