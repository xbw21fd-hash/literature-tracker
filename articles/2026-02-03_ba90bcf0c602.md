## Abstract
arXiv:2507.11520v2 Announce Type: replace Abstract: Many empirical systems contain complex interactions of arbitrary size, representing, for example, chemical reactions, social groups, co-authorship relationships, and ecological dependencies. These interactions are known as higher-order interactions and the collection of these interactions comprise a higher-order network, or hypergraph. Hypergraphs have established themselves as a popular and versatile mathematical representation of such systems and a number of software packages written in various programming languages have been designed to analyze these networks. However, the ecosystem of higher-order network analysis software is fragmented due to specialization of each software's programming interface and compatible data representations. To enable seamless data exchange between higher-order network analysis software packages, we introduce the Hypergraph Interchange Format (HIF), a standardized format for storing higher-order network data. HIF supports multiple types of higher-order networks, including undirected hypergraphs, directed hypergraphs, and abstract simplicial complexes, while actively exploring extensions to represent multiplex hypergraphs, temporal hypergraphs, and ordered hypergraphs. To accommodate the wide variety of metadata used in different contexts, HIF also includes support for attributes associated with nodes, edges, and incidences. This initiative is a collaborative effort involving authors, maintainers, and contributors from prominent hypergraph software packages. This project introduces a JSON schema with corresponding documentation and unit tests, example HIF-compliant datasets, and tutorials demonstrating the use of HIF with several popular higher-order network analysis software packages.

## 摘要（中文）
arXiv:2507.11520v2 公告类型：替换 摘要：许多经验系统包含任意大小的复杂相互作用，例如化学反应、社会群体、合着关系和生态依赖性。这些交互被称为高阶交互，这些交互的集合构成了高阶网络或超图。超图已成为此类系统的流行且通用的数学表示形式，并且已设计了许多用各种编程语言编写的软件包来分析这些网络。然而，由于每个软件的编程接口和兼容的数据表示的专业化，高阶网络分析软件的生态系统是支离破碎的。为了实现高阶网络分析软件包之间的无缝数据交换，我们引入了超图交换格式（HIF），这是一种用于存储高阶网络数据的标准化格式。 HIF 支持多种类型的高阶网络，包括无向超图、有向超图和抽象单纯复形，同时积极探索表示多重超图、时间超图和有序超图的扩展。为了适应不同上下文中使用的各种元数据，HIF 还支持与节点、边和关联相关的属性。该计划是由著名超图软件包的作者、维护者和贡献者共同努力的结果。该项目引入了带有相应文档和单元测试的 JSON 模式、符合 HIF 的数据集示例，以及演示如何使用 HIF 与几种流行的高阶网络分析软件包的教程。
