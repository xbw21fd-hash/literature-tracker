# Learning from Discriminatory Training Data

## Abstract
arXiv:1912.08189v5 Announce Type: replace-cross Abstract: Supervised learning systems are trained using historical data and, if the data was tainted by discrimination, they may unintentionally learn to discriminate against protected groups. We propose that fair learning methods, despite training on potentially discriminatory datasets, shall perform well on fair test datasets. Such dataset shifts crystallize application scenarios for specific fair learning methods. For instance, the removal of direct discrimination can be represented as a particular dataset shift problem. For this scenario, we propose a learning method that provably minimizes model error on fair datasets, while blindly training on datasets poisoned with direct additive discrimination. The method is compatible with existing legal systems and provides a solution to the widely discussed issue of protected groups' intersectionality by striking a balance between the protected groups. Technically, the method applies probabilistic interventions, has causal and counterfactual formulations, and is computationally lightweight - it can be used with any supervised learning model to prevent direct and indirect discrimination via proxies while maximizing model accuracy for business necessity.

## 摘要（中文）
arXiv:1912.08189v5 公告类型：replace-cross 摘要：监督学习系统使用历史数据进行训练，如果数据受到歧视的污染，它们可能会无意中学会歧视受保护群体。我们建议，尽管在潜在歧视性数据集上进行了训练，公平学习方法仍应在公平测试数据集上表现良好。这样的数据集转变具体化了特定公平学习方法的应用场景。例如，直接歧视的消除可以表示为特定的数据集转移问题。对于这种情况，我们提出了一种学习方法，可以证明最大限度地减少公平数据集上的模型误差，同时对受到直接加性歧视的数据集进行盲目训练。该方法与现有法律体系兼容，通过在受保护群体之间取得平衡，为广泛讨论的受保护群体交叉性问题提供了解决方案。从技术上讲，该方法应用概率干预，具有因果和反事实公式，并且计算量轻 - 它可以与任何监督学习模型一起使用，以防止通过代理进行直接和间接歧视，同时最大限度地提高模型的准确性以满足业务需求。
