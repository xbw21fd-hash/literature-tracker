## Abstract
arXiv:2512.12786v2 Announce Type: replace Abstract: Solar energetic particle (SEP) events pose severe threats to spacecraft, astronaut safety, and aviation operations. Accurate SEP forecasting remains a critical challenge in space weather research due to their complex origins and highly variable propagation. In this work, we built SEPNET, an innovative multi-task neural network that jointly predicts future solar eruptive events, including solar flares and coronal mass ejections (CMEs) and SEPs, incorporating long short-term memory and transformer architectures that capture contextual dependencies. SEPNet is a machine learning framework for SEP prediction that utilizes an extensive set of predictors, including solar flares, CMEs, and space-weather HMI active region patches (SHARP) magnetic field parameters. SEPNET is rigorously evaluated on the SEPVAL SEP dataset (Whitman, 2025b), which is used to evaluate the performance of the current SEP prediction models. The performance of SEPNet is compared with classical machine learning methods and current state-of-the-art pre-eruptive SEP prediction models. The results show that SEPNET, particularly with SHARP parameters, achieves higher detection rates and skill scores while maintaining suitable for real-time space weather alert operations. Although class imbalance in the data leads to relatively high false alarm rates, SEPNET consistently outperforms reference methods and provides timely SEP forecasts, highlighting the capability of deep multi-task learning for next-generation space weather prediction. All data and code are available on GitHub at https://github.com/yuyian/SEP-Prediction.git.

## 摘要（中文）
arXiv:2512.12786v2 公告类型：替换 摘要：太阳高能粒子（SEP）事件对航天器、宇航员安全和航空运营构成严重威胁。由于其复杂的起源和高度可变的传播，准确的 SEP 预报仍然是空间天气研究中的一个关键挑战。在这项工作中，我们构建了 SEPNET，这是一种创新的多任务神经网络，可以联合预测未来的太阳爆发事件，包括太阳耀斑和日冕物质抛射 (CME) 和 SEP，结合了长短期记忆和捕获上下文依赖性的变压器架构。 SEPNet 是用于 SEP 预测的机器学习框架，它利用了一组广泛的预测变量，包括太阳耀斑、CME 和空间天气 HMI 活动区域补丁 (SHARP) 磁场参数。 SEPNET 在 SEPVAL SEP 数据集（Whitman，2025b）上进行了严格评估，该数据集用于评估当前 SEP 预测模型的性能。将 SEPNet 的性能与经典机器学习方法和当前最先进的喷发前 SEP 预测模型进行了比较。结果表明，SEPNET，特别是具有 SHARP 参数的 SEPNET，在保持适合实时空间天气警报操作的同时，实现了更高的检测率和技能得分。尽管数据中的类别不平衡导致相对较高的误报率，但SEPNET始终优于参考方法并提供及时的SEP预报，凸显了深度多任务学习在下一代空间天气预报中的能力。所有数据和代码均可在 GitHub 上获取：https://github.com/yuyian/SEP-Prediction.git。
