## Abstract
arXiv:2510.25966v2 Announce Type: replace Abstract: Delayed generalization, termed grokking, in a machine learning calculation occurs when the increase in test accuracy is delayed relative to the training accuracy. This paper examines grokking in the context of a dense neural network trained to classify 2D Ising model configurations into 4 equally spaced energy regions in the presence of weight decay. Partially with the aid of novel PCA-based network layer analysis techniques, the observed behavior is interpreted as a transition from a connected network to a group of sparse subnetworks in which the number of active weights in each layer decreases monotonically with depth. This architecture reduces classification errors resulting from a multiplicity of paths. The final network layers, as in a convolutional neural network, sequentially identify global features of the input classes, which enables generalization to previously unseen patterns.

## 摘要（中文）
arXiv:2510.25966v2 公告类型：替换 摘要：在机器学习计算中，当测试精度的增加相对于训练精度延迟时，就会出现延迟泛化（称为 grokking）。本文在密集神经网络的背景下研究了 grokking，该网络经过训练，可在存在权重衰减的情况下将 2D Ising 模型配置分类为 4 个等距的能量区域。部分借助基于 PCA 的新型网络层分析技术，观察到的行为被解释为从连接网络到一组稀疏子网络的过渡，其中每层中的活动权重的数量随着深度单调减少。这种架构减少了由多个路径导致的分类错误。最后的网络层，就像在卷积神经网络中一样，顺序识别输入类的全局特征，这使得能够泛化到以前未见过的模式。
