## Abstract
arXiv:2512.23880v1 Announce Type: cross Abstract: Large language model (LLM) agents currently depend on predefined tools or brittle tool generation, constraining their capability and adaptability to complex scientific tasks. We introduce CASCADE, a self-evolving agentic framework representing an early instantiation of the transition from "LLM + tool use" to "LLM + skill acquisition". CASCADE enables agents to master complex external tools and codify knowledge through two meta-skills: continuous learning via web search and code extraction, and self-reflection via introspection and knowledge graph exploration, among others. We evaluate CASCADE on SciSkillBench, a benchmark of 116 materials science and chemistry research tasks. CASCADE achieves a 93.3% success rate using GPT-5, compared to 35.4% without evolution mechanisms. We further demonstrate real-world applications in computational analysis, autonomous laboratory experiments, and selective reproduction of published papers. Along with human-agent collaboration and memory consolidation, CASCADE accumulates executable skills that can be shared across agents and scientists, moving toward scalable AI-assisted scientific research.

## 摘要（中文）
arXiv:2512.23880v1 公告类型：交叉 摘要：大型语言模型（LLM）代理目前依赖于预定义工具或脆弱的工具生成，限制了它们对复杂科学任务的能力和适应性。我们介绍 CASCADE，一个自我进化的代理框架，代表了从“LLM + 工具使用”到“LLM + 技能获取”过渡的早期实例。 CASCADE 使代理能够掌握复杂的外部工具并通过两种元技能将知识编码：通过网络搜索和代码提取进行持续学习，以及通过内省和知识图探索进行自我反思等。我们在 SciSkillBench 上评估 CASCADE，SciSkillBench 是 116 项材料科学和化学研究任务的基准。 CASCADE 使用 GPT-5 实现了 93.3% 的成功率，而没有使用进化机制的成功率为 35.4%。我们进一步展示了在计算分析、自主实验室实验和已发表论文的选择性复制方面的实际应用。除了人类与智能体协作和记忆整合之外，CASCADE 还积累了可以在智能体和科学家之间共享的可执行技能，从而朝着可扩展的人工智能辅助科学研究迈进。
