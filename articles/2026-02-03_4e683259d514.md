## Abstract
arXiv:2602.01269v1 Announce Type: new Abstract: In this study, we evaluated the performance of SAIPy, an open-source Python package for deep learning-based seismic data analysis, by applying its single-station monitoring tools and extending its use to a seismic network based approach, using data from a local seismic network deployed in a Caldera. Although the integrated models into SAIPy for earthquake detection,magnitude estimation, seismic phase picking, and P-wave polarity classification, were originally trained on tectonic signals, we assess their performance in a more complex seismic environment that includes volcano-tectonic events, along with signal interference from distant earthquakes.We also demonstrate the advantages of integrating outputs using multiple stations to improve event detection. SAIPy was able to identify a significantly larger number of local events than those included in previously published catalogs. SAIPy demonstrated reliable phase picking and P-wave polarity estimation, particularly for local volcano-tectonic events, with some limitations observed in the magnitude estimation for complex volcanic signals. These results support the utility of SAIPy for processing continuous seismic data and suggest that future retraining using data with physically standardized units, removing instrumental response, and including data from more diverse seismic sources, could improve its generalization for magnitude estimation to complex scenarios and different seismic networks and sensor types.

## 摘要（中文）
arXiv:2602.01269v1 公告类型：新 摘要：在这项研究中，我们评估了 SAIPy 的性能，SAIPy 是一个用于基于深度学习的地震数据分析的开源 Python 包，通过应用其单站监测工具并将其用途扩展到基于地震网络的方法，使用来自火山口部署的本地地震网络的数据。尽管 SAIPy 中用于地震检测、震级估计、地震相位拾取和 P 波极性分类的集成模型最初是针对构造信号进行训练的，但我们在更复杂的地震环境中评估了它们的性能，包括火山构造事件以及来自遥远地震的信号干扰。我们还展示了使用多个站集成输出以改进事件检测的优势。 SAIPy 能够识别出比之前发布的目录中包含的本地事件数量要多得多的本地事件。 SAIPy 展示了可靠的相位拾取和 P 波极性估计，特别是对于局部火山构造事件，但在复杂火山信号的震级估计中观察到了一些局限性。这些结果支持 SAIPy 在处理连续地震数据方面的实用性，并表明未来使用具有物理标准化单位的数据进行再训练、去除仪器响应并包含来自更多样化地震源的数据，可以提高其对复杂场景和不同地震网络和传感器类型的震级估计的泛化能力。
