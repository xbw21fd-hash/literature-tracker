## Abstract
arXiv:2602.10121v1 Announce Type: new Abstract: In coastal volcanic aquifers, the reliability of freshwater seawater-exchange simulations are governed by accuracy of the conceptual groundwater model (CGM). The traditional CGMs are constructed by qualitatively combining independent hydrogeophysical features, limiting their ability to capture the complexity of volcanic terrains. To integrate these disparate, sparse, and imbalanced features, we propose an AI-assisted workflow. First, the self-organizing map (SOM) is applied to estimate a deterministic set of transdisciplinary features called the reference model. Second, generative algorithms are applied to the reference model and empirical distributions constructed to obtain sets of stochastic point clouds called the site model. Data quality metrics identify the preferred generative algorithm whose set of stochastic features are mapped using SOM to the groundwater model grid and assigned as the stochastic CGM. The proposed algorithm is applied to extremely imbalanced multiclass features and multiple discrete numerical features observed at the Halawa-Moanalua aquifer, Oahu, Hawaii. At this stie, the Copula Generative Adversarial Network is deemed as the preferred generative algorithm whose set of stochastic transdisciplinary features represent the Halawa-Moanalua CGM. The simulated spatial geologic units correspond to published surface maps; and the simulated conductance, temperature, and barometric pressure profiles correlate with those measured at deep monitoring wells. Inspecting the 3-dimensional conductance models reveal groundwater flow and discharge driven by the aquifer hydraulic gradient, freshwater pumping, seawater intrusion induced by onshore withdrawals, and preferred pathways for freshwater-seawater exchange, such as landward intrusion of seawater and seaward discharge of freshwater.

## 摘要（中文）
arXiv:2602.10121v1 公告类型：新 摘要：在沿海火山含水层中，淡水海水交换模拟的可靠性取决于概念地下水模型 (CGM) 的准确性。传统的 CGM 是通过定性地结合独立的水文地球物理特征来构建的，限制了它们捕捉火山地形复杂性的能力。为了整合这些不同、稀疏和不平衡的特征，我们提出了人工智能辅助的工作流程。首先，应用自组织映射 (SOM) 来估计一组确定性的跨学科特征，称为参考模型。其次，将生成算法应用于构建的参考模型和经验分布，以获得称为站点模型的随机点云集。数据质量指标确定首选生成算法，其随机特征集使用 SOM 映射到地下水模型网格，并指定为随机 CGM。所提出的算法适用于在夏威夷瓦胡岛 Halawa-Moanalua 含水层观察到的极度不平衡的多类特征和多个离散数值特征。在这一点上，Copula 生成对抗网络被认为是首选生成算法，其随机跨学科特征集代表了 Halawa-Moanalua CGM。模拟的空间地质单元与已发布的地表图相对应；模拟的电导、温度和气压剖面与深部监测井测量的数据相关。检查 3 维电导模型揭示了含水层水力梯度驱动的地下水流和排放、淡水抽水、陆上抽取引起的海水入侵，以及淡水-海水交换的首选途径，例如海水向陆地入侵和淡水向海排放。
